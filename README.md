# Zachary Music (zachmusic2)

music aistudio

Overview

This repository contains the Zachary Music aistudio frontend and server. It is a TypeScript-based project with a small static frontend (HTML/CSS) and a Node server.

Quick start

1) Install dependencies

```bash
npm install
```

2) Create environment file

Copy the example and provide your secret values:

```bash
cp .env.example .env
# then edit .env
```

Do NOT commit your real .env file to the repo.

3) Development

```bash
npm run dev
```

4) Build

```bash
npm run build
```

5) Start (production build)

```bash
npm start
```

Environment

See `.env.example` for the variables required by the project. Keep all server-side keys secret and use your deployment provider's secret store (Vercel/Netlify/GitHub Actions secrets) when deploying.
